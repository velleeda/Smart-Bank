# How to run

```
# Clone this repository
$ git clone https://github.com/velleeda/Smart-Bank

# Go to the directory
$ cd Smart-Bank

# Install Dependencies
$ npm install

# Run Web Server
$ npm start

# Run Server
$ npm run server
```
